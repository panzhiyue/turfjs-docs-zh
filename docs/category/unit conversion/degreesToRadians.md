# degreesToRadians

> Converts an angle in degrees to radians
> 将角度转换为弧度

```text
> npm install @turf/helpers
```

**参数**

| 参数    | 类型   | 描述                            |
| :------ | :----- | :------------------------------ |
| degrees | number | angle between 0 and 360 degrees |

**返回**

number - angle in radians

```js
turf.degreesToRadians(60); // 1.0471975511965976
```
